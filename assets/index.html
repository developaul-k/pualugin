Toggle

- CODE: 기본

HTML 

  <div data-element="toggle">
    <button class="pualugin__button" data-element="toggle__anchor">toggle_anchor</button>
    <div data-element="toggle__panel">toggle_panel</div>
  </div>

- RESULT

toggle_panel

- CODE: 옵션 추가

HTML 

  <div data-element="toggle" data-options='{ "isOpened": true, "onChangeBeforeText": "열기", "onChangeAfterText": "닫기" }'>
    <button class="pualugin__button" data-element="toggle__anchor">toggle_anchor</button>
    <div data-element="toggle__panel">toggle_panel</div>
  </div>

- RESULT

toggle_panel
Options
  • mode
    • type: String
    • default: static
    • description: static, slide, fade 중 선택
  • isOpened
    • type: Boolean
    • default: false
    • description: 초기에 toggle__panel 노출 여부를 설정
  • event
    • type: String
    • default: click
    • description: click, focusin 중 선택
  • easing
    • type: String
    • default: swing
  • activeClassName
    • type: String
    • default: is-active
    • description: 토글 활성화 클래스
  • onChangeBeforeText
    • type: String
    • default: null
    • description: panel 오픈 전 노출될 anchor 텍스트
  • onChangeAfterText
    • type: String
    • default: null
    • description: panel 오픈 후 노출될 anchor 텍스트

- Event

JAVASCRIPT 

  var yourVariable = $('.your-element');

  yourVariable.on('beforeChange', function( event, plugin, anchor, panel ){
      /* your code ... */
  })
  
  yourVariable.on('afterChange', function( event, plugin, anchor, panel ){
      /* your code ... */
  })
  • beforeChange
    • arguments: event, plugin, anchor, panel
    • description: 패널 오픈 전 실행될 함수
  • afterChange
    • arguments: event, plugin, anchor, panel
    • description: 패널 오픈 후 실행될 함수

- Method

JAVASCRIPT 

  var yourVariable = $('.your-element').data('plugin_toggle');

  yourVariable.show();

  yourVariable.hide();

  yourVariable.destroy();

  yourVariable.reInit();
  • show
    • arguments: none
    • description: 패널 열기
  • hide
    • arguments: none
    • description: 패널 닫기
  • destroy
    • arguments: none
    • description: 플러그인 제거
  • reInit
    • arguments: none
    • description: 플러그인 제거 후 생성

- 모든 플러그인은 기본적으로 페이지 렌더링시 Initialized 됩니다. 개별적(비동기 마크업 구현) Initialize 필요한 경우 아래와 같이 진행합니다.

JAVASCRIPT 

  $('.your-element').toggle(); // Initialize
