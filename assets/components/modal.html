Plugin - Modal

- CODE / 기본

HTML

  <!-- 모달 버튼 -->
  <button class="pualugin__button" data-element="modal__open" data-target="#modal">모달 오픈</button>
  <!-- 모달 -->
  <div id="modal" class="pualugin-modal__element" data-element="modal__element">
    <div class="pualugin-modal__container" data-element="modal__element-container">
      <div class="pualugin-modal__contents" data-element="modal__element-contetns">
        모달 내용입니다.
        <button class="pualugin-modal__close" data-element="modal__close">닫기</button>
      </div>
    </div>
  </div>

- RESULT

모달 내용입니다.
Options
  • modalWidth
    • type: Number
    • default: 500
  • modalHeight
    • type: Number
    • default: 500
  • activeClassName
    • type: String
    • default: is-open

- Event

JAVASCRIPT 

  var yourVariable = $('.your-element');

  yourVariable.on('beforeChange', function( event, plugin, modal ){
      /* your code ... */
  })

  yourVariable.on('afterChange', function( event, plugin, modal ){
      /* your code ... */
  })
  • beforeChange
    • arguments: event, plugin, modal
    • description: 모달 오픈 전 실행될 함수
  • afterChange
    • arguments: event, plugin, modal
    • description: 모달 오픈 후 실행될 함수

- Method

JAVASCRIPT 

  var yourVariable = $('.your-element').data('plugin_modal');

  yourVariable.open();

  yourVariable.close();

  yourVariable.destroy();

  yourVariable.reInit();
  • open
    • arguments: none
    • description: 모달 열기
  • close
    • arguments: none
    • description: 모달 닫기
  • destroy
    • arguments: none
    • description: 플러그인 제거
  • reInit
    • arguments: none
    • description: 플러그인 제거 후 생성

- 모든 플러그인은 기본적으로 페이지 렌더링시 Initialized 됩니다. 개별적(비동기 마크업 구현) Initialize 필요한 경우 아래와 같이 진행합니다.

JAVASCRIPT 

  $('.your-element').modal(); // Initialize
