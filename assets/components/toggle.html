Toggle

- CODE - 기본

HTML 

  <div data-js="toggle">
    <button class="pualugin__button" data-js="toggle__anchor">toggle_anchor</button>
    <div data-js="toggle__panel">toggle_panel</div>
  </div>

- RESULT

toggle_panel

- CODE / 옵션 : isOpened: true, onChangeBeforeText: 열기, onChangeAfterText: 닫기

HTML 

  <div data-js="toggle" data-options='{ "isOpened": true, "onChangeBeforeText": "열기", "onChangeAfterText": "닫기" }'>
    <button class="pualugin__button" data-js="toggle__anchor">toggle_anchor</button>
    <div data-js="toggle__panel">toggle_panel</div>
  </div>

- RESULT

toggle_panel

- 모든 플러그인은 기본적으로 페이지 렌더링시 Initialized 됩니다. 개별적(비동기 마크업 구현) Initialize 필요한 경우 아래와 같이 진행합니다.

JAVASCRIPT 

  $('.your-element').toggle(); // Initialize
Options
  • mode
    • type: String
    • default: static
    • description: static, slide, fade 중 선택
  • isOpened
    • type: Boolean
    • default: false
    • description: 초기에 toggle__panel 노출 여부를 설정
  • event
    • type: String
    • default: click
    • description: click, focusin 중 선택
  • easing
    • type: String
    • default: swing
  • activeClassName
    • type: String
    • default: is-active
    • description: 토글 활성화 클래스
  • onChangeBeforeText
    • type: String
    • default: null
    • description: panel 오픈 전 노출될 anchor 텍스트
  • onChangeAfterText
    • type: String
    • default: null
    • description: panel 오픈 후 노출될 anchor 텍스트
